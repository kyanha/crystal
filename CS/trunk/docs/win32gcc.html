Crystal Space installation with GCC for Windows

Written by Seth Galbraith

Andrew Zabolotny, myself and others are working on instructions for using Crystal Space with a GNU compiler for windows, but it isn't easy because of the many different versions of the GNU Windows compiler and problems with some Windows header files containing non-standard code used by Microsoft. If you're interested in helping out with this project, contact me or one of the other Crystal Space developers.

Files you will need

full.exe               - Cygwin GNU Windows compiler
winlibs14.zip          - compiled zip, jpeg, and png libraries for Windows
nasm-0.98e-bin-w32.zip - NASM (Netwide Assembler) for Windows

Installation

  • Unzip the compiled libraries into the same folder where you unzipped Crystal Space. (See Bugs.)
  • Unzip NASM somwhere.
  • Run the Cygwin full.exe installer.
  • To use GCC and NASM you need to make sure you have the appropriate bin directories in your PATH. Here's my own PATH as an example: 
    SET PATH=C:\Cygnus\cygwin-b20\H-i586-cygwin32\bin;C:\nasm\bin\win32;%PATH%
  • Now just CD to your Crystal Space directory, run make and follow the instructions.

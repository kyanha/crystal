How to handle alpha.

Enumeration values:
alphaNone  Ignore alpha.
alphaBinary  Binary alpha (pixels with alpha >0.5 are drawn, all others not).
alphaSmooth  'Smooth' alpha (colors are mixed based on a pixel's alpha value)

Detailed Description

template<class T>
class csFmtDefaultReader< T >

Standard format string source for csPrintfFormatter.

Remarks:
Uses csUnicodeTransform::Decode(), so it supports utd8_char, utf16_char, utf32_char and wchar_t strings.

enum csDialogFrameStyle
 

Possible dialog frame styles.

Dialog can have no frame (default) or can have a horizontal or vertical "frame" which is used for horizontal and vertical toolbars.

Enumeration values:
csdfsNone  Dialog has no frame (default).
csdfsHorizontal  Dialog has two 3D horizontal lines at top and bottom (like menu bar).
csdfsVertical  Dialog has two 3D vertical lines at left and right.
csdfsAround  Dialog has a 3D frame around perimeter.
csdfsBitmap  Dialog's frame and background are defined entirely by a bitmap.

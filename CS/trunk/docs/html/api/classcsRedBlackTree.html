Detailed Description

template<typename K>
class csRedBlackTree< K >

A red-black-tree.

Remarks:
Does not allow duplicate keys.

Uses csComparator<> for key comparisons.

Only stores keys. If you need a key-value-map, look at csRedBlackTreeMap.

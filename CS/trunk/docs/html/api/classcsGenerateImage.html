Detailed Description

This class will compute a texture for a terrain.

The texture is based on the heightmap for the terrain. It is like the povray MaterialMap, but then indexed with the height of the terrain. This means, that given some base textures that should display at certain heights, the whole texture is computed. At each pixel the base textures are blended together, based on the height.

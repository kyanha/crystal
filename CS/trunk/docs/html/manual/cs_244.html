7.11.4 AWS Standard Sink Trigger Reference

The following table lists triggers that the `awsStandardSink' hosts.

Hide
Hides the component that sent the signal.

Show
Shows the component that sent the signal.

Invalidate
Invalidates the component that sent the signal (forces a re-draw).

HideWindow
Hides the window containing the component that sent the signal.

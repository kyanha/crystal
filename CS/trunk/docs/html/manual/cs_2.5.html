2.5 Platform Specific Instructions

When working with Crystal Space, always keep in mind that this is a multi-platform project. Sometimes not all features will work correctly or be available on all platforms. If you are using the CVS version of Crystal Space and run across a problem, it usually pays off to post a message to the Crystal Space mailing list describing you problem along with relevant diagnostic messages. See section Mailing Lists.

For some ports there is additional reference information explaining the internal workings of the port. When such detailed information exists, you will find a reference to it from within the instructions in this section.

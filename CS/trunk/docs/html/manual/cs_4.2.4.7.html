4.2.4.7 Lighting Commands

It is possible to add dynamic lights at your current location. These have a serious degradation on frame rate, but can be fun to play with.

addlight r,g,b,radius,shadows [Shift-L or 1]

Create dynamic light of specified colour at current location. Shadows is 0 or 1 to cast shadows. Defaults are radius=6, colour =white. If no parameters are given the light will also move up and down slightly and randomly change colour. The rgb components should be floating point numbers in the range 0.0 to 1.0, but this is not checked.

dellight [Shift-1]

Delete most recently added dynamic light.

dellights [Alt-1]

Delete all dynamic lights.

clrlights

Set all pseudo-dynamic lights to black.

setlight r, g, b

Sets the selected light to specified colour. The selected light is the most recently added.

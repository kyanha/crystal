4.2.2 Mouse Movement

The mouse is used for pointing at things, not movement, unless the "freelook" option is set on. In this case, moving the mouse steers the camera, and Button 1 moves forward.

  • Button 1 (normally the left one) is forward movement.

  • Button 2 (the middle one if present) describes the nearest Mesh Object to where the cursor is.

  • Button 3 (normally the right one) gives the Camera and World coordinates of where the mouse cursor is, and if a polygon is under the cursor, the name if any is given.

These actions can be changed. See section 4.2.6 Configuration Files.

7.11.5.3 Label

A label's type is "Label".

Signals

Labels send the following signals.

signalLabelClicked
Occurs when the user clicks a mouse button on the component.

Attributes

Labels publish the following attributes.

Caption
The text of the label.
Align
The alignment of the label.
  • lblAlignLeft: align label text left
  • lblAlignCenter: align label text in the center of the control
  • lblAlignRight: align label text right

Properties

Labels publish the following properties.

Caption
The text of the label.

C.1.21 Model Importing

The old model importing system has been completely removed. That means the following interfaces and classes are gone:

  • csSingleIndexVertexSet
  • csModelDataTools
  • csModelConverterFormat
  • iModelConverter
  • iModelDataTexture
  • iModelDataMaterial
  • iModelDataVertices
  • iModelDataAction
  • iModelDataPolygon
  • iModelDataObject
  • iModelDataCamera
  • iModelDataLight
  • iModelData
  • csModelDataTexture
  • csModelDataMaterial
  • csModelDataVertices
  • csModelDataAction
  • csModelDataPolygon
  • csModelDataObject
  • csModelDataCamera
  • csModelDataLight
  • csModelData
  • iCrossBuilder

Instead it is now possible to load 3DS models as genmesh mesh objects and MD2 models as 3D sprites. See section Loading 3DS and MD2 Models.

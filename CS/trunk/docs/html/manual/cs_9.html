1.8 Internet Relay Chat (IRC)

The Crystal Space developers are often available online for live chat via the Internet Relay Chat (IRC) system. If you are unfamiliar with IRC, a good place to learn about it is http://irchelp.org/. The Crystal Space developers inhabit a channel named #CrystalSpace on the FreeNode Network. You can find more information about FreeNode at http://freenode.net/, including a list of servers. There are a few redirection servers to which you should be able to connect right away. Try connecting to one located closest (geographically) to you for the least lag.

 
irc.au.freenode.net
irc.cn.freenode.net
irc.eu.freenode.net
irc.us.freenode.net

7.5.9 3D Sprites

Written by Seth Galbraith, sgalbrai@linknet.kitsap.lib.wa.us.

3D sprites are triangle-based models which can be animated by pre-calculated vertex animations. This type of model is well suited for detailed models which are not easily represented as heightmapped terrain or bezier curves. 3D sprites can be used in your application through the mesh object plugin systems (i.e. a 3D sprite is one particular type of mesh object). See section Mesh Object Plug-In System.

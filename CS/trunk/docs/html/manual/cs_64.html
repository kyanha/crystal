4.3 Quake Map Converter (map2cs)

Written by Thomas Hieber, thieber@gmx.net. Last updated 19 March 2000.

Currently, Crystal Space has reached a state where it can be very rewarding to create some maps for Crystal Space. That's where `map2cs' comes in handy. It allows you to convert maps created with mature Quake map building tools to Crystal Space's map format.

People new to `map2cs' may be interested in the accompanying tutorial (see section 5.5 map2cs Tutorial), and all users (new and old) may benefit from the accompanying reference information which follows.

1.2 Current Status

Note: Crystal Space (CS) is still in beta. This means that there are known and unknown bugs and that things in general may not work as expected (or may not work at all). For information about known bugs and unfinished items, refer to the Developer's Project page at SourceForge:

http://sourceforge.net/projects/crystal/

There are already a number of projects using Crystal Space. For more information, please visit the Related Projects page:

http://crystal.sourceforge.net/tikiwiki/tiki-index.php?page=Related+Projects

Crystal Space includes both a hardware (OpenGL) and software 3D renderer. For decent performance in the software renderer, a reasonably modern machine should be used. With an OpenGL 3D card, Crystal Space can really shine.

There is a very active list of developers working to make Crystal Space as usable as possible, so things are improving constantly.

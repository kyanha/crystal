5.7.3 Step 3: Creating a Home Directory

Before setting up WinCVS, itself, you need to create a home directory somewhere on your system. The home directory is a place where WinCVS will store some CVS administrative information, such as CVS passwords, servers you login too etc. For the purposes of this tutorial, it is assumed that you have created the home directory named `C:\home'.

4.19 Alternate Windowing System 2 (AWS2)

Written by Christopher Nelson, nadiasvertex@gmail.com.

The Alternate Windowing System 2 is a plugin designed to replace the older Alternate Windowing System (AWS).

It uses a vector pen instead of the older bitmap graphics-centric idea, but also allows shapes to be texture mapped so that scalable bitmaps can in fact be used.

The vast majority of AWS2 is written in JavaScript, but a native-code drawing and event layer makes sure that the system is not too slow. It also uses a caching layer so that per-frame redraws are very optimal.

For example code, examine projects which utilize AWS2:

  • CS/apps/tools/partedit2
  • CS/apps/tests/awstest

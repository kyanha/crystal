5.5.1 Step 1: Links to Required Tools and Data

To get started creating your own maps, we assume you have downloaded and installed the following tools from the Internet. WorldCraft 3.3 and `map2cs' require a quite complex setup, which we shall explain later in this tutorial.

Links to Required Tools and Data

  • Crystal Space - This sounds obvious, but you require a release distribution of the Crystal Space SDK. You can obtain it from:

    http://crystal.sourceforge.net/

  • Valve Hammer 3.4 - The Half-Life editor. You might be technically or legally required to have a legal Half-Life on your PC too, so you might want to get this really cool game too, if you don't have it yet.

    http://collective.valve-erc.com/

  • Texture Pack 1 - More than 150 free textures from the Crystal Space texture archive in 3.4MB. They are already converted to the WAD format, so you can directly use them.

    ftp://sunsite.dk/projects/crystal/support/map2cs/cstex_1.zip

  • Texture Pack 2 - More than 250 free textures from the Crystal Space texture archive in 8.6MB. They are already converted to the WAD format, so you can directly use them.

    ftp://sunsite.dk/projects/crystal/support/map2cs/cstex_2.zip

Links to Optional and Alternative Tools

  • QuArK - Another editor. Has a bit strange license, but is mostly Open Source. We don't use that editor, but it has some massive fans, and it looks really nice. It offers a more flexible approach for texturing, than Worldcraft, using a proprietary extension to the map format. These extensions are understood by `map2cs'. You only need this editor if you don't want to use Worldcraft 3.3, but this tutorial will only cover Worldcraft 3.3.

    http://www.planetquake.com/quark/

  • Wally - A great tool to modify or create textures for use with Worldcraft and Crystal Space. Required only if you want to create new textures.

    http://www.telefragged.com/wally/

We assume you will use the following structure to install your programs. You can use any structure you like, but for the purposes of this tutorial it will be assumed you have the following directory structure:

 
C:\Crystal
    CS
    Maps
    Quark
    WADs
    Wally
    WorldCraft

Here is a quick summary of each of the subdirectories:

CS - Root directory of the Crystal Space distribution.

Maps - Put your own or downloaded or created map files (`.map' and `.rmf') here.

Quark - If you wish to use Quark, install it here.

Wads - Put all texture files (`.wad' and `.pk3') here.

Wally - If you wish to use Wally, install it here.

WorldCraft - Install WorldCraft here.

5.5.2 Step 2: Configuring Worldcraft

Configuring WorldCraft sometimes is a pain. But there is one consolation: Configuring WorldCraft for use with Half-Life isn't any fun either.

  • Extract the texture pack(s) which you have downloaded. Put the extracted `.wad' file into a directory on the same drive, as you have stored WorldCraft. (You can choose any directory. We will assume, you are using `C:\Crystal\Wads' further on.

  • Just after you started Worldcraft for the first time, you will be prompted with a dialog that requires some settings for it. (If you accidently closed that dialog, you can call it again via the Tools/Options menu item.)

These following are the settings we use.

The "General" settings tab.

The "2D views" settings tab.

The "3D views" settings tab.

The "Textures" settings tab.

In the "Texture Files" list, add the `.wad' files which you will be using. (For instance, `D:\Crystal\Wads\cstex_1.wad' from our example.)

The "Game Configurations" settings tab. Fill in the fields based upon the following table:

Field Value
Configuration Crystal Space
Game Data Files `C:\Crystal\WorldCraft\halflife.fgd'

The "Build Programs" settings tab.

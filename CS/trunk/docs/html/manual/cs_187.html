7.6.7.8 In-sector Visibility

For in-sector visibility either `frustvis' or `dynavis' can be used. `frustvis' only does frustum culling so we will discuss `dynavis' here.

Dynavis is roughly based on `dPVS' (http://www.hybrid.fi/dpvs_download.html) but at this moment we are not that advanced yet :-) I recommend you read the reference document you can download from there. It is very interesting.

5.5 Making Levels and Models in 3DSMax

3DSMax is a very powerful commercial 3D modeller which is supported very well by Crystal Space.

Making Levels

For making levels you can use the exporter that is included with Crystal Space. You can find it in `CS/scripts/max'. There is also a manual there which is located at `CS/scripts/max/exporterguide/index.html'.

Making Models

Models of type Cal3d (see section SpriteCal3D Mesh Object) can be converted from 3DSMax.

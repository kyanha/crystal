5.6.4 Step 4: Creating a Simple Room in Worldcraft

The following steps may seem trivial for somebody who knows Worldcraft in and out. So all the Worldcraft pros just skip to the next step.

  1. Open Worldcraft.
  2. Select File/New.
  3. Select the "Block" tool either by pressing SHIFT-b or selecting it from the toolbar.

  4. Left-click in of of the 2D windows, hold the button down and drag the mouse until you have created a box of reasonable size. Release the mouse button once completed.
  5. Now you should see a block made of dashed lines on every 2D window that displays some handles. Drag these handles until the block has a size of 512-W by 512-L by 512-H (shown in the status bar). Center the block around the origin (the crossing of the blue lines).
  6. When the block has proper size, press ENTER. The handles should then disappear, and the lines should become solid.
  7. Switch to the "Selection" tool either by pressing SHIFT-s or by selecting it from the toolbar.

  8. Select the block by clicking at one of its border lines. The handles should appear again. (You could change size again if you like by dragging a handle.)
  9. Invoke the Tools/Make Hollow menu item and select a width of 32 units. Now you should have a hollow cube.
  10. Select the "Entity" tool. (SHIFT-e or select it from the toolbar.)

  11. Select "light" in the dialog on the right side:

  12. Click in the top right view inside the cube, now use one of the bottom views to drag it to a position near the top.
  13. Press ENTER. A small pink square should appear.
  14. Select the square by left-clicking on it. After it is selected use a right-click and select "Properties" from the drop down menu. (Due to a bug in WorldCraft, you are forced to change to properties for every light source; otherwise it is completely dark.)
  15. In the "Properties" dialog select "Brightness" and "Pick Color" select a white color. (The light is defined by 4 values. The first three components specify color and the last component specifies brightness / radius. Normally a radius of 200 is okay.)

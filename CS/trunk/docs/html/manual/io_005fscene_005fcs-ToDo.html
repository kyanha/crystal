5.8.7 ToDo

This section lists the main features of the exporter that are still lacking. Please contact the Crystal Space mailing list if you wish to help and add support for them.

  • Limited support for camera properties
  • Limited support for material properties (Could use defining custom shader variables)
  • No support for terrains (WIP, alphamap generation is working based on blender stencil maps, heightmap generation is WIP)
  • No support for particles
  • No support for sounds
  • Limited support for physics properties
  • No support for iBodySkeleton
  • No support for trees of armatures: only the root armature is exported, not the children ones

5.7.5 Step 5: Setting Location to Checkout Crystal Space

This section is all about personal preference, however the actions remain the same wherever you choose to checkout the Crystal Space source code too on your system -- so here is an example. First, create a directory on your `C:\' drive called `crystal_space'. Now from the file menu of WinCVS click on View then Browse Location and finally Change. You will be presented with a browse dialog. Just point it to the `crystal_space' directory on `C:\' which you just created and then click on the OK button. If all is well, the directory `crystal_space' should appear at the top of the left hand Workspace window of the WinCVS client.

Example: 

 CS_IDENT_STRING_LIST(FooNames)
   CS_IDENT_STRING(FOO_BAR)
   CS_IDENT_STRING(FOO_BAZ)

From code, use like: 
   csPrintf ("%s\n", FooNames.StringForIdent (foo));

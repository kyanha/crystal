float qisqrt (  float    x )  [inline, static]
 

This routine is basically equivalent to qsqrt() except that it returns 1/sqrt(x) rather than the proper square root.

It should be used anywhere you need the inverse root (in 3D graphics it is a common situation), because the routine is a little faster than qsqrt() and also you avoid a division.

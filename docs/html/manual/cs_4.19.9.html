4.19.9 XML AWS2 Interface Objects

AWS2 uses a sophisticated and flexible means to define user interfaces in XML.

The version of Javascript, currently 1.6, used by AWS2 includes a specification called "ECMAscript for XML" or E4X for short. All interfaces are designed using this specification.

4.3.3 Invoking

`map2cs' doesn't really have many options, but they should be mentioned anyway:

 
map2cs mapfile worldfile [configfile]

You must specify the name of the mapfile and the worldfile. (You can use relative paths here if you want.)

You can optionally override the configuration file by specifying configfile. If you don't specify an alternate configuration file, `map2cs.cfg' is being used.

Example:

 
map2cs ..\maps\mymap.map data\mymap.map mymyp.cfg

5.5.1 Installing io_scene_cs

Installation procedure:

Check Dependencies

  • CrystalSpace for viewing worlds. Better if it's the last version. Instructions to download and install CrystalSpace are available here.
  • Python for export. It is usually included in Linux distributions, but must be downloaded by Windows users.
  • Blender for modeling 3D worlds and objects. io_scene_cs 1.0 requires Blender 2.59 or later release.

Download

Download the scripts through SVN:

 
svn co https://crystal.svn.sourceforge.net/svnroot/crystal/CS/trunk/scripts/blender  blender

Install

Manually install the Add-On by copying the folder ‘blender/io_scene_cs’ into Blender Add-Ons directory (‘blender/$version/scripts/addons’).

Set Environment Variables

The ‘CRYSTAL’ environment variable must be pointing to the CrystalSpace directory. Although it's not technically needed to use io_scene_cs and export, you'll need it to actually view your exported files and to use the ‘Export and Run’ functionality.

Enable Add-On

Add-Ons are found in the ‘Add-Ons’ tab of the ‘Files > User Preferences’ window. Once an Add-On has been installed, it has to be enabled before it can be used. Simply place a check mark on the ‘Enable an addon’ box of the Add-On called ‘Import-Export: Export to CrystalSpace 3D format’.

The new export functionality is now integrated into Blender and can be used. If you want to disable the functionality, uncheck the box. To get more information on the Add-On you can press the arrow at the left of the entry. Finally, this Add-On can be enabled everytime you start Blender simply by saving your user preferences.

Test

Reload or open Blender and check if the CS export panel appears in the ‘Render’ context of the ‘Properties’ window.

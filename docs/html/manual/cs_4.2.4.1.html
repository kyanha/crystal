4.2.4.1 Playing the Game

Before we start explaining the game logic itself, first a little bit of information on gameplay. If you start the `mazing' game that is included with Crystal Space you can use the following keys to operate the game:

  • `a': stafe left.
  • `d': stafe right.
  • `w': forward.
  • `s': backward.
  • `q': move up.
  • `e': move down.
  • `space': fire laser.
  • `arrow up': rotate camera up.
  • `arrow down': rotate camera down.
  • `arrow left': rotate camera left.
  • `arrow right': rotate camera right.
  • `esc': quit game.

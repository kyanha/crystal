4.2.6 Configuration Files

Walktest has its own configuration file, `/data/config/walktest.cfg'. This can be used to set the default map directory, override other plugin defaults and also contains many settings for the Collision Detection to prevent the player from walking through things.

In addition, Walktest reads the normal configuration files for each plugin. Probably the one of most interesting is the VFS configuration file, at `/vfs.cfg'. You will need to alter this if you want to load a map not otherwise known to VFS, or add a `VFS.Mount.lev' line to your own configuration file somewhere.

Finally, Walktest reads `/data/config/autoexec.cfg', which contains a series of Walktest commands. Normally this only contains key binding commands, but can in principle contain any Walktest commands.

5.3 Making Levels and Models in Blender

Blender is one of the most popular free 3D modellers available and it is supported very well by Crystal Space. You can find Blender here:

http://www.blender3d.org

Making Levels

For making levels there are basically two ways. Either you can use the Blend2cs utility which is a standalone application that can convert a blender file to a Crystal Space world file. You can get it from:

  • Official project site: http://projects.blender.org/projects/blend2cs/
  • More recent windows compiled version: ftp://sunsite.dk/projects/crystal/crystalcore/blend2cs.exe and a readme for that at: ftp://sunsite.dk/projects/crystal/crystalcore/README_blend2cs.txt.

Another way is to use the blender2crystal exporter which can be integrated directly with Blender itself. Download and more information are available here:

http://b2cs.delcorp.org/index.php/Main_Page

Making Models

Models of type Genmesh (see section Genmesh Mesh Object) can also be converted directly with Blend2cs or blender2crystal. To do this you need to mark the object in Blender as a genmesh object (see manual on how to do that) and then export as a library. In Crystal Space you can directly load that library as a single model.

Models of type Cal3d (see section SpriteCal3D Mesh Object) can be converted from Blender using the Cal3d exporter that is included with Blender.

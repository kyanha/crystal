6. Facilities

This chapter describes the features of the facilities which are linked into typical Crystal Space applications and plugins. For convenience, all of these facilities are contained in the primary `crystalspace' library, so it is not strictly necessary to distinguish them for actual usage, however, for developers working on Crystal Space itself, it is helpful to understand how these facilities are broken down and the relationship between them.

6.1 Utility Module (csUtil)  
6.2 Shared Class Facility (SCF)  
6.3 Geometry Module (csGeom)  

This section presents a quick overview of the facilities provided by Crystal Space. More detailed descriptions of these facilities may be found in later sections.

cstool
This facility contains various high-level classes and utilities. They tend to be much more specialized than the general-purpose tools provided elsewhere and may even depend upon plugin modules (unlike the other facilities mentioned here).

csutil
This facility contains many low-level utilities including string and hash classes, to ZIP manipulation tools, plugin management and Unicode support modules.

SCF (Shared Class Facility)
SCF is actually a component of "csutil", however, it is explained separately since it is both complex (compared to the rest of "csutil") and fundamental to the understanding of plugin modules. See section 6.2 Shared Class Facility (SCF).

csgeom
A facility for geometric operations. It provides 2D and 3D vectors, polygons and bounding boxes, 3D matrices and some other useful tools. See section 6.3 Geometry Module (csGeom).

csgfx
Basic image support and graphical manipulation tools.

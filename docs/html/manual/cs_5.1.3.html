5.1.3 Compiling the Application

In order to compile a Crystal Space application I recommend you look at the howto on compiling an external application. For MSVC 6 this tutorial can be found at See section Creating an External MSVC 6 Application, for other environments based on GNU make you can look at See section Creating an External Crystal Space Application.

Additionally note that the source for some of these tutorials can be found in the `CS/apps/tutorials' directory. There are also a few additional tutorials there that are not (yet) covered in the documentation.
